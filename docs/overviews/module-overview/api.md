`api` 模块是simbot最基础的模块，对绝大部分基础内容进行定义约束。

## 命名空间
说是命名空间，实际上只是“包路径”的定义方式。


- `love.forte.simbot.*`

此路径下（不含深层级路径）的内容一般是一些通用内容，比如 [唯一标识](../../defition/ID.md)


- `love.forte.simbot.event`

一切与事件调度相关的内容。比如事件拦截器、事件过滤器、事件执行器、事件管理器、监听函数等。
