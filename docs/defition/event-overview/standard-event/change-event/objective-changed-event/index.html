对象变更事件

对象变更事件。代表为一个 Objectives 发生了变化的事件。

MemberChangedEvent​

api.member_changedChangedEventOrganizationEventMemberEvent

代表一个组织中发生的 成员变动 事件。通常表现为 成员增加 或 成员减少 。

属性类型描述
operatorMemberInfo?成员变动的操作者。如果本次变动行为发生的原因在于本次变动的成员自身，则此属性值等同于此成员；在不支持、不存在等情况下可能为null。

MemberIncreaseEvent​

api.member_increaseIncreaseEventMemberChangedEvent

代表一个组织中增加了一个成员。

属性类型描述
afterMember增加的组织成员
actionTypeActionType行为类型

MemberDecreaseEvent​

api.member_decreaseDecreaseEventMemberChangedEvent

代表一个组织中减少了一个成员。

属性类型描述
beforeMember离开的组织成员
actionTypeActionType行为类型

ActionType​

枚举类型，代表为一个行为的类型。

元素描述
PROACTIVE主动的
PASSIVE被动的

FriendChangedEvent​

api.friend_changedChangedEventFriendEvent

代表一个bot的好友（数量）发生了变化。

属性类型描述
sourceBot发生好友变化的bot
friendFriend变化的好友对象

FriendIncreaseEvent​

api.friend_increaseIncreaseEventFriendChangedEvent

代表一个bot的好友增加了。

属性类型描述
sourceBot发生好友变化的bot
friendfriend增加的好友对象
after同上同上

FriendDecreaseEvent​

api.friend_decreaseDecreaseEventFriendChangedEvent

代表一个bot的好友减少了。

属性类型描述
sourceBot发生好友变化的bot
friendfriend减少的好友对象
before同上同上
