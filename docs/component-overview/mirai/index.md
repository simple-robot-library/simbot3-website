### 组件优势

- 对接十分优秀的**第三方开源框架 [Mirai](https://github.com/mamoe/mirai)**，社区活跃度高、功能强大。
- 第三方框架封装度与灵活度高，降低了组件的开发难度，更加简单易用。

:::

:::danger 😟

### 组件劣势

- 非官方API，稳定性与可靠性**无法保证**。
- 部分内容受限于依赖框架的更新与维护。

:::
